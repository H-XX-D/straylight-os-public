## Responsibilities

Maintainers are responsible for:

- Reviewing public pull requests.
- Keeping release hygiene checks current.
- Separating verified behavior from alpha, experimental, or gated work.
- Maintaining public documentation and examples.
- Keeping private infrastructure details out of the public repository.

## Contact

Use GitHub issues and pull requests for public repository work. Use the
repository security policy for security-sensitive reports.
