# Support

This repository is an alpha public starter snapshot for StrayLight OS. It is
published for documentation, examples, packaging layout, release hygiene, and
source-snapshot collaboration.

## What Is Supported Here

Use GitHub issues for:

- Documentation defects.
- Missing or unclear build and ISO requirements.
- Sanitized example requests.
- Release hygiene or CI problems.
- Public package-layout questions.

Use pull requests for:

- Documentation fixes.
- Sanitized examples.
- Release-check improvements.
- Public packaging or build guidance improvements.

## What Is Not Supported Here

This public repository cannot support:

- Private lab infrastructure debugging.
- Logs containing private hostnames, addresses, user paths, hardware IDs, or
  credentials.
- Unpublished source trees or local workstation state.
- Production support commitments.
- Binary ISO or package artifacts not attached to an explicit verified release.

## Reporting Security Issues

Follow [SECURITY.md](SECURITY.md). Do not put credentials, exploit payloads,
private host data, or unredacted logs in public issues.

## Alpha Expectations

Treat this repository as a public starter and release-hygiene frame. Hardware,
kernel, networking, installer, and ISO paths should be considered alpha or gated
unless the relevant validation is documented in the release notes.
